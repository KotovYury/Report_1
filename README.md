# Report_1
### Входные параметры:
Файл Excel, на основе которого необходимо составить отчет.
### Выходные параметры:
Файл Excel, используемый для дальнейшей работы.
### Описание работы робота:
При запуске робота пользователь выбирает файл Excel, на основе которго будет составлен требуемый отчет, ипользуемый для дальнейшей работы. Значение месяца, информация за который содержится в выбранном файле, будет получено из названия файла. Далее создается структура финальной таблицы, которая заполняется только необходимыми данными из тех, которые получены из исходного файла. Так же производится заполнение некоторых недостающих данных,  которые в 99% случаев имеют одно и то же повторяющееся значение. Так как исходные данные приходят ежемесячно, а финальный отчет составляется поквартально, то на основе значения месяца, полученного вначале, произойдет создание и сохранение нового файла с расширением xlsx, в названии которого будет указан номер квартала, в который входит этот месяц, если это первый месяц этого квартала, либо добавление данных к уже ранее созданному файлу, если это второй либо третий месяц квартала.

Робот разработан на платформе UiPath с применением языка программирования C# для работы с переменными.
